My Pet-projects

Weather App

Pixel-art themed Weather app built with HTML5, CSS3 and Vanilla JS. The app displays the main information about the current temperature, short description of weather conditions, humidity percentage, wind speed and a five day forecast in a searched city. The app is integrated using OpenWeatherMap API. Bootstrap 5 is used to make the app responsive.

Features:

  • User can switch between Celcius and Fahrenheit units with the units button;
  • Dynamic background which changes according to the part of the day;
  • Dynamic animation for a better representation of weather conditions in the region;

Code is open-sourced and can be viewed on GitHub.
